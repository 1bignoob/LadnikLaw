# Law Website

A professional legal services website built with HTML, CSS, and JavaScript.

## Project Structure

```
Law/
├── index.html      # Main HTML file with page structure
├── styles.css      # Stylesheet for page layout and design
├── script.js       # JavaScript for interactivity
└── README.md       # This file
```

## Features

- Responsive design that works on desktop and mobile
- Sticky navigation header
- Smooth scrolling between sections
- Professional color scheme suited for legal services
- Multiple sections: Home, About, Services, and Contact

## Getting Started

1. Open `index.html` in your web browser
2. Navigate using the menu links
3. Customize content in `index.html` as needed
4. Modify styling in `styles.css`
5. Add more interactivity in `script.js`

## Customization

- **Colors**: Update the color values in `styles.css` to match your brand
- **Content**: Edit the text and sections in `index.html`
- **Functionality**: Add JavaScript features in `script.js`

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
